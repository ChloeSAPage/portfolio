# Get it running locally

### 1. Clone the Repository

Using Git, copy these files into a folder.
Use this command:

```
git clone https://github.com/ChloeSAPage/portfolio.git
```

### 2. Install the requirements

```
npm install
```

### 3. Run

```
npm run dev
```
